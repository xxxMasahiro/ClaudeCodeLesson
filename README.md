# TODO アプリ

📌 設計・実装・公開までを AI と協業しながら一貫して行うことを意識しました。


フレームワークを使わず、純粋な HTML / CSS / JavaScript で実装した TODO 管理アプリケーションです。
Claude Code を活用し、設計から実装、GitHub Pages での公開まで一通りを実践したポートフォリオプロジェクトです。

---

**実際に動作するデモはこちらからご覧いただけます**
👉 **[https://xxxmasahiro.github.io/ClaudeCodeLesson/](https://xxxmasahiro.github.io/ClaudeCodeLesson/)**
※ ブラウザのみで動作します（インストール不要）

---

## 📋 主な機能

- **TODO の追加・編集・削除**：完全な CRUD 操作を実装
- **完了状態の管理**：チェックボックスによる直感的な操作
- **データ永続化**：localStorage を使用し、ブラウザを閉じてもデータを保持
- **レスポンシブ対応**：モバイルからデスクトップまで快適に使用可能
- **アクセシビリティ対応**：キーボードナビゲーションとセマンティック HTML を採用

---

## 🛠 使用技術

| カテゴリ | 技術 |
|---------|------|
| フロントエンド | HTML5 / CSS3 / JavaScript (ES6+) |
| ビルドツール | Vite |
| データ永続化 | localStorage API |
| デプロイ | GitHub Pages |

### 採用した実装パターン

- **バニラ JavaScript**：フレームワークに依存しないシンプルな構成
- **DOM 操作最適化**：DocumentFragment、イベント委譲による効率化
- **モジュール分割**：storage / app の責務を明確に分離
- **セキュリティ対策**：XSS 防止のため `textContent` を使用、入力バリデーション実装

## 💻 ローカルでの起動方法

### 前提条件

- Node.js がインストールされていること

### セットアップ手順

```bash
# リポジトリをクローン
git clone https://github.com/xxxmasahiro/ClaudeCodeLesson.git
cd ClaudeCodeLesson

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスすると動作確認できます。

### ビルド

```bash
# 本番用ビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

## 📐 制作で意識したポイント

### 1. パフォーマンス
- DocumentFragment を使った効率的な DOM 操作
- イベント委譲による最小限のイベントリスナー登録
- CSS 変数を活用したスタイル管理

### 2. ユーザビリティ
- Enter / Escape キーによるキーボード操作対応
- 完了状態の視覚的フィードバック（取り消し線、透明度）
- TODO 数のカウンター表示

### 3. 保守性
- 責務ごとのモジュール分割（storage.js / app.js）
- エラーハンドリングの適切な実装
- 明確な命名規則とコード構造
