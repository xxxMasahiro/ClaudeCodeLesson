# TODO アプリ

シンプルで学習に最適なTODOアプリケーションです。フレームワークを使わず、純粋なHTML/CSS/JavaScriptで実装されています。

## 特徴

- **バニラJavaScript**: フレームワークなし、純粋なJavaScriptで実装
- **データ永続化**: localStorageを使用してデータを保存
- **完全なCRUD操作**: 作成、読取、更新、削除の全機能
- **レスポンシブデザイン**: モバイルからデスクトップまで対応
- **モダンなUI**: CSS変数、flexbox、アニメーションを活用
- **アクセシビリティ**: セマンティックHTML、キーボードナビゲーション対応

## 機能

### 基本機能
- ✅ TODOの追加
- ✅ TODOの表示
- ✅ 完了状態のトグル
- ✅ TODOの編集（インライン編集）
- ✅ TODOの削除
- ✅ データの永続化（ページを更新しても保持）

### UI機能
- 📊 TODOカウンター（合計数・完了数）
- 🎨 完了状態の視覚的フィードバック（取り消し線、透明度）
- ✨ スムーズなアニメーション
- 📱 レスポンシブデザイン
- 🌟 ホバーエフェクト

## 使い方

### インストール

このアプリケーションはインストール不要です。ファイルをダウンロードするだけで使用できます。

```bash
# リポジトリをクローン
git clone <repository-url>
cd ClaudCodeLesson
```

### 起動方法

1. `index.html` をブラウザで開くだけです
2. または、VSCodeのLive Serverなどを使用することもできます

```bash
# 簡易サーバーを起動する場合（Python 3）
python -m http.server 8000

# ブラウザで http://localhost:8000 にアクセス
```

### 使用方法

1. **TODOの追加**
   - 上部の入力フィールドにタスクを入力
   - 「追加」ボタンをクリック、またはEnterキーを押す

2. **TODOの完了**
   - チェックボックスをクリックして完了/未完了を切り替え

3. **TODOの編集**
   - 「編集」ボタンをクリック
   - テキストを変更してEnterキーで確定
   - Escapeキーでキャンセル

4. **TODOの削除**
   - 「削除」ボタンをクリック
   - 確認ダイアログで「OK」を選択

## データ構造

各TODOは以下の構造で保存されます：

```javascript
{
  id: "1704258000000abc123",  // ユニークID
  text: "買い物をする",        // TODO内容
  completed: false,           // 完了状態
  createdAt: 1704258000000,   // 作成日時（ミリ秒）
  updatedAt: 1704258000000    // 更新日時（ミリ秒）
}
```

## トラブルシューティング

### TODOが保存されない

- ブラウザのlocalStorageが有効になっているか確認
- プライベートブラウジングモードでは保存されません
- ストレージ容量が一杯の可能性があります

### スタイルが適用されない

- CSSファイルのパスが正しいか確認
- ブラウザのキャッシュをクリア

### 動作しない

- JavaScriptファイルのパスが正しいか確認
- ブラウザのコンソールでエラーを確認
- ブラウザが最新版か確認
